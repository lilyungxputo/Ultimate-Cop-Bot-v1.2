Keywords

Documentation

Keywords is the main feature of this bot. Before each drop (every Thursday at 11 AM EST), a photo of the drop with keywords/colors will be available here. (keywords are not sure at 100 %, it’s only suppositions). You choose the category of item then you write keywords and color wanted. You can also choose the size. If it does not matter to you, the option "No matter" will choose a random available size.
You can add keywords for multiple items. Please check your keywords before start the bot each time.

To start using keywords, go to supremenewyork.com, then click on the extension logo on top-right of your browser and click on "Start" button, please choose right option you want on "Main settings" tab before.

Once bot is started, it will check if keywords are in item name. If yes, item will be added to cart. Please see our video to see how does it work.
